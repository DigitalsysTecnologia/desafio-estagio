# AdventureRealm

AdventureRealm é um projeto de página inicial para um jogo fictício que combina elementos de vikings e cavaleiros templários em uma batalha épica. Este repositório contém o código fonte para a página inicial do jogo, desenvolvido com React e estilizado utilizando o framework Chakra UI.

## Resumo

A página inicial do AdventureRealm apresenta diversas seções que convidam os visitantes a explorar o mundo do jogo, conhecer suas missões, e descobrir os personagens envolvidos nesta aventura épica. Com uma combinação de imagens de fundo, texto descritivo e componentes interativos, a página proporciona uma introdução imersiva ao universo do jogo.

## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para construir interfaces de usuário.
- **Chakra UI**: Framework de design de componentes React que facilita a criação de interfaces consistentes e bonitas.
- **framer-motion**: Biblioteca para animações em React.
- **react-icons**: Conjunto de ícones populares para React.
- **react-router-dom**: Biblioteca de roteamento para React.
- **react-slick**: Componente de carrossel para React.
- **slick-carousel**: Biblioteca jQuery para carrossel, utilizada em conjunto com react-slick.
- **styled-components**: Biblioteca para estilização de componentes com CSS-in-JS.

## Como Rodar em Sua Máquina

1. Clone este repositório para sua máquina local utilizando o seguinte comando no terminal:

    ```bash
    git clone https://github.com/seu-usuario/adventurerealm.git
    ```

2. Navegue até o diretório do projeto:

    ```bash
    cd adventurerealm
    ```

3. Instale as dependências do projeto utilizando o npm (ou yarn, se preferir):

    ```bash
    npm install
    ```

4. Após a instalação das dependências, inicie o servidor de desenvolvimento:

    ```bash
    npm start
    ```

5. O aplicativo será iniciado automaticamente no seu navegador padrão. Se isso não acontecer, você pode acessá-lo manualmente digitando `http://localhost:3000` na barra de endereços do seu navegador.
